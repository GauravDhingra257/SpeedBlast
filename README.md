## Installation

Install Speed Blast

```bash
  Clone project to your local machine 💻
  Open browser and go to Manage extensions ⚙
  Enable developer mode 👨‍💻
  Click load unpacked 🔃
  Select the speedBlast/Speed-Blast folder ⚡
  Extension is successfully installed  🔥
```
